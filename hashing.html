Hashing

The technique of turning a given key into another value is known as hashing. A mathematical technique is employed to produce the new value using a hash function. A hash value, or simply a hash, is the output of said function. A good hash function utilizes a one-way hashing method, which means that the hash cannot be transformed back into the original key.

Applications of hashing

The most typical application of hashing is to implement hash tables. A hash table is a list that holds key/value pairs in which any entry may be retrieved by its index. We may use a hash function to map the keys to the size of the table since there is no limit to the number of pairs; the hash value becomes the index for a specific item.

Cryptography also makes use of hashing. Passwords can be kept in the form of hashes, such that unencrypted credentials are not available even if a system is compromised. Popular encryption hashes include MD5, SHA and AES.
